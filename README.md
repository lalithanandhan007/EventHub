# EventHub – Online Event Registration System

EventHub is a web-based event registration and management system.

Features:
- View upcoming events
- Register for events
- Backend API using Node.js and Express
- Dashboard displaying registered users

Technologies Used:
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

How to Run:
1. Navigate to backend folder
2. Run: node server.js
3. Open index.html using Live Server
